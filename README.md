# HybridRAIDReconstructor

This tool is a tool to rebuild RAID against Linux based hybrid RAID

* support hybrid RAID
 - Drobo, BeyondRAID
 - NETGEAR, X-RAID2
 - Seagate, SimplyRAID
 - Synology, SHR

* how to use
 - input : only raw image file. input the directory path.
 - output : create extract image file. input the directory path.
